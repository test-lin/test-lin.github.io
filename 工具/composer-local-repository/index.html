1. github 的包需要配置 token

解决方法：

访问命令行中提示的 https://github.com/settings/tokens/new?scopes=repo&description=Composer+on+packages.example.org+2018-06-28+0310

复制 token description 文本框中内容

拉到页底 点击 generate token

在命令行中粘贴复制内容确认限可
